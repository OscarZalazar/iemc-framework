# Roadmap

This roadmap describes the public, open-source direction for IEMC Framework. It does not describe private OZD Engine or ONE ADS implementation plans.

## Phase 1: Documentation

- Publish the initial framework definition.
- Define core terminology for voice, text, visual, narrative, and cultural signals.
- Create synthetic examples.
- Document public-safe contribution guidelines.

## Phase 2: Example Datasets

- Create small synthetic datasets for demonstration.
- Add annotation guidelines.
- Publish data cards describing assumptions and limitations.
- Avoid real user data and private business data.

## Phase 3: Benchmark

- Define public benchmark tasks for multimodal emotional interpretation.
- Include cases for irony, humor, cultural context, and ambiguous intent.
- Compare context-free and context-aware interpretation strategies.
- Publish reproducible evaluation templates.

## Phase 4: Public SDK

- Explore a minimal SDK for public research workflows.
- Keep the SDK separate from proprietary systems.
- Provide adapters only for open examples and public datasets.
- Include transparent outputs and audit-friendly traces.

## Phase 5: Research Community

- Invite academic and independent research contributions.
- Collect multilingual and multicultural evaluation scenarios.
- Publish discussion papers and benchmark reports.
- Develop community governance for public research artifacts.
